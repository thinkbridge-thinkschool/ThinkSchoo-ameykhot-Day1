⚠ Dirty Read: Session B reads Balance = 99999.00 from Session A's open (uncommitted) transaction. After Session A rolls back, that value never existed.
SESSION A Runs first — updates but does NOT commit
USE IsolationDemo;
SET TRANSACTION ISOLATION LEVEL READ UNCOMMITTED;

BEGIN TRANSACTION;
    UPDATE Accounts SET Balance = 99999.00 WHERE Id = 20;
    -- transaction deliberately kept open
    WAITFOR DELAY '00:00:10';
ROLLBACK;
-- dirty value erased; actual Balance = 4000.00
