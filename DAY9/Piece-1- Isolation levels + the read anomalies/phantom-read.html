thinkschool-sql-amey.database.windows.net  |  thinkschool-quotesdb-amey  |  Isolation Level Demo — ANOMALY 3: PHANTOM READ
⚠️  Phantom Read: Session 2 runs the same range query twice. Session 1 inserts a new row in between. Session 2 sees a different row count — a "phantom" row appeared.
SESSION 2  READ COMMITTED — COUNT(*) changes between two identical queries
SET TRANSACTION ISOLATION LEVEL READ COMMITTED; BEGIN TRANSACTION; -- First count SELECT COUNT(*) AS QuoteCount FROM Quotes WHERE AuthorId = 1; WAITFOR DELAY '00:00:05'; -- window for S1 to INSERT -- Second count (identical query) SELECT COUNT(*) AS QuoteCount FROM Quotes WHERE AuthorId = 1; COMMIT;
